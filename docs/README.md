## 기능목록📋

1. [X] 다리 길이를 입력받는다.


2. [X] 다리 길이 입력값을 검증한다.
   1. [X] 숫자인지 검증한다.
   2. [X] 3 이상 20 이하의 숫자인지 검증한다.


3. [X] 다리를 생성한다.
   1. [X] 0 과 1 중 랜덤으로 한 값을 생성한다.
   2. [X] 다리의 길이만큼 반복한다.


4. [X] 건널 위치를 입력받는다.


5. [X] 위치 입력값을 검증한다.
   1. [X] "U" 혹은 "D" 인지 검증한다.


6. [X] 지도를 생성한다.
   1. [X] 올바른 위치인 경우 "O"로 표시한다.
   2. [X] 틀린 위치인 경우 "X"로 표시한다.


7. [X] 지도를 출력한다.


8. [X] 게임 재시작 여부를 입력받는다.


9. [X] 재시작 여부 입력값을 검증한다.
   1. [X] "R" 혹은 "Q" 인지 검증한다.


10. [X] 게임 결과를 출력한다.
    1. [X] 최종 지도를 출력한다.
    2. [X] 시도 횟수를 출력한다.
    3. [X] 성공 여부를 출력한다.
