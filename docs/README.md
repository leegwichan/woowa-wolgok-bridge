## 기능목록📋

1. [ ] 다리 길이를 입력받는다.


2. [ ] 다리 길이 입력값을 검증한다.
   1. [ ] 숫자인지 검증한다.
   2. [ ] 3 이상 20 이하의 숫자인지 검증한다.


3. [ ] 다리를 생성한다.
   1. [ ] 0 과 1 중 랜덤으로 한 값을 생성한다.
   2. [ ] 다리의 길이만큼 반복한다.


4. [ ] 건널 위치를 입력받는다.


5. [ ] 위치 입력값을 검증한다.
   1. [ ] "U" 혹은 "D" 인지 검증한다.


6. [ ] 지도를 출력한다.
   1. [ ] 올바른 위치인 경우 "O"로 표시한다.
   2. [ ] 틀린 위치인 경우 "X"로 표시한다.


7. [ ] 게임 재시작 여부를 입력받는다.


8. [ ] 재시작 여부 입력값을 검증한다.
   1. [ ] "R" 혹은 "Q" 인지 검증한다.
